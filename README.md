# E-Commerce Sales & Customer Analytics Platform

## Project Overview

This project analyzes e-commerce sales data to uncover business insights related to sales performance, customer behavior, profitability, product performance, and regional trends.

The project follows a real-world analytics workflow including data cleaning, exploratory data analysis (EDA), SQL reporting, spreadsheet analytics, and Power BI dashboarding.

## Business Insights

### Sales Performance

- Total Sales: $2.29M
- Total Profit: $286K
- Total Orders: 5009

### Customer Insights

- Total Customers: 793
- Consumer segment generated highest revenue.
- Top Customer: Sean Miller

### Product Insights

- Technology category generated highest sales.
- Canon imageCLASS 2200 Advanced Copier was the top-selling product.

### Regional Insights

- West region generated highest revenue.
- California was the best-performing state.

### Profitability Insights

- High discounts negatively impacted profitability.
- Several products generated significant losses.
